Sejarah HTML: Dari 1991 sampai Sekarang

HTML punya perjalanan panjang sebelum menjadi fondasi web modern seperti sekarang. Yuk kita lihat evolusinya.

  1. Diciptakan Tahun 1991 oleh Tim Berners-Lee
    Ilmuwan asal Inggris ini bikin HTML saat bekerja di CERN (Swiss). Tujuannya biar para peneliti bisa saling berbagi dokumen dengan mudah. Awalnya cuma ada 18 tag sederhana kayak <h1>, <p>, <a> (tautan).
  2. HTML 2.0 (1995) - Standar Resmi Pertama
    Ini versi pertama yang resmi distandarisasi. Mulai ada fitur formulir <form>.
  3. HTML 3.2 (1997) - W3C Ambil Alih
    W3C (konsorsium web) mulai mengurus HTML. Versi ini nambahin dukungan buat tabel, applet Java, dan teks yang bisa membungkus gambar.
  4. HTML 4.01 (1999) - Puncak HTML Klasik
    Versi ini ngenalin konsep penting: pisahin struktur (HTML) sama tampilan (CSS). Tag kayak <font> mulai ditinggalin. Versi ini paling lama dipake, sampai bertahun-tahun.
  5. XHTML (2000-2006) - Jalan Buntu
    W3C coba bikin HTML jadi lebih ketat ala XML. Harus tutup tag, pakai huruf kecil semua. Tapi terlalu ribet dan nggak praktis, akhirnya gagal dan ditolak developer.
  6. Lahirnya HTML5 (2004-2014)
    Karena frustrasi sama W3C yang lambat, kelompok dari Apple, Mozilla, Opera bikin WHATWG dan mulai ngembangin HTML versi baru. Tahun 2014, HTML5 resmi jadi standar. Ini lompatan gede banget.

Revolusi HTML5

Yang bikin HTML5 beda:

  • Elemen semantik: <header>, <footer>, <article> biar struktur lebih jelas.
  • Multimedia native: <video> dan <audio> tanpa perlu Flash.
  • Canvas: Bisa gambar dan bikin game pake JavaScript.
  • Formulir pintar: Input type email, date, number, dll.
  • API canggih: Geolocation, penyimpanan lokal, web workers.

Sekarang: Standar Hidup

Mulai 2019, HTML nggak punya versi besar lagi. Dia jadi "living standard" yang terus diperbarui step by step sesuai kebutuhan.

Intinya

HTML lahir tahun 1991 buat kebutuhan akademis, berkembang pelan sampai HTML 4, sempet nyasar di XHTML, lalu direvolusi HTML5 jadi platform aplikasi web modern, dan sekarang terus berevolusi tanpa versi.
