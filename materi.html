Apa Itu HTML?

Lu pernah buka website kan? Nah, di balik semua tampilan keren itu, ada kerangka dasar yang namanya HTML. Singkatannya HyperText Markup Language. Ribet? Iya, tapi konsepnya gampang.

HTML itu bahasa markup, BUKAN bahasa pemrograman. Bedanya apa? Bahasa pemrograman kayak JavaScript atau Python tuh ngurusin logika, hitung-hitungan, dan bikin sesuatu terjadi. Sementara HTML cuma nandain konten doang.

Coba bayangin lu lagi ngetik di Word. Saat lu blok tulisan "Ini Judul" terus lu ganti jadi "Heading 1", itu gunanya sama kayak HTML. Lu lagi ngasih tau dokumen bahwa tulisan itu penting dan harus jadi judul. Bedanya, di HTML kita pake kode-kode kecil.

Caranya gampang, kita pake yang namanya tag. Bentuknya gini <nama-tag> buat pembuka, dan </nama-tag> buat penutup. Di antara tag itu baru isi kontennya.

Contoh paling basic (coba jalankan di editor bawah):

<h1>Ini Judul Utama</h1>
<p>Ini paragraf biasa.</p>
<a href="https://google.com">Klik ke Google</a>

<a> di atas adalah contoh tautan. Atribut href menentukan alamat tujuannya.

Terus HTML tuh kerangka doang?

Iya, persis. HTML cuma ngurus struktur aja. Biar gampang, inget analogi orang:

  • HTML = tulang kerangka. Yang nentuin mana kepala, mana badan, mana tangan.
  • CSS = kulit dan baju. Yang ngatur warna, ukuran, tampilan biar cakep.
  • JavaScript = otot. Yang bikin bisa gerak dan interaktif, kayak pas tombol diklik muncul pop-up.

Struktur minimal HTML kayak gimana?

Setiap halaman HTML mesti punya kerangka wajib gini:

<!DOCTYPE html>
<html>
  <head>
    <title>Judul Website</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini halaman pertama gue.</p>
  </body>
</html>

Nah, gampang kan maksudnya:

  • <!DOCTYPE html> cuma ngasih tau browser bahwa ini pake HTML versi terbaru (HTML5).
  • <html> itu bungkus besar yang isi semua konten.
  • <head> itu bagian kepala. Isinya data-data penting yang nggak keliatan langsung, kayak judul halaman (<title>) yang muncul di tab browser.
  • <body> ini baru bagian inti. Semua yang mau lu liat di halaman web, dari teks, gambar, video, semua harus ditaruh di sini.

Kesimpulan

HTML itu fondasi dasar dari semua website. Dia yang bikin tulisan bisa jadi judul, jadi paragraf, jadi link. Tanpa HTML, browser cuma liat teks acak-acakan tanpa struktur. Mau website sekelas Google, YouTube, atau cuma blog pribadi, semuanya pake HTML sebagai tulang punggungnya.

Udah, itu aja intinya. Nggak serumit yang dibayangin, cuma perlu hafal dan rajin latihan aja.
