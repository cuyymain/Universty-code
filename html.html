Apa Itu HTML?

HyperText Markup Language, bahasa standar untuk membuat halaman web.

Sejarah HTML

Diciptakan oleh Tim Berners-Lee pada 1991, berkembang hingga HTML5.

Versi HTML

HTML 1.0, 2.0, 3.2, 4.01, XHTML, HTML5, Living Standard.

DOCTYPE

Deklarasi tipe dokumen, memberitahu browser versi HTML yang digunakan.

Struktur Dasar HTML

<!DOCTYPE>, <html>, <head>, <body>.

Elemen HTML

Segala sesuatu dari tag pembuka hingga tag penutup.

Atribut HTML

Informasi tambahan pada elemen, ditulis di tag pembuka.

<html>

Elemen root dari dokumen HTML.

<head>

Kontainer untuk metadata (data tentang data).

<title>

Judul halaman yang tampil di tab browser.

Menentukan URL dasar untuk semua relative URL.
